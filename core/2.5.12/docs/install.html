Instalace XOOPS

První instalace

Úvod:

XOOPS je open-source objektově orientovaný systém pro publikování na webu napsaný v PHP. Je ideálním nástrojem pro vývoj malých až velkých dynamických komunitních webových stránek, interních firemních portálů, korporátních portálů, weblogů a mnoho dalšího.

XOOPS je vydáván pod podmínkami GNU General Public License (GPL) verze 2 nebo vyšší a je zdarma k použití a úpravám. Je zdarma k redistribuci, pokud dodržíte distribuční podmínky GPL.

Požadavky

  • WWW server (Apache, NGINX, IIS atd.)
  • PHP 7.4 nebo vyšší, doporučuje se 8.3.0+
  • MySQL 5.7 nebo vyšší, doporučuje se MySQL 8.4.5+

Před instalací

  1. Nastavte WWW server, PHP a databázový server správně.
  2. Připravte databázi pro váš XOOPS web.
  3. Připravte uživatelský účet a udělte uživateli přístup k databázi.
  4. Zpřístupněte adresáře a soubory pro zápis: %s
  5. Z bezpečnostních důvodů se důrazně doporučuje přesunout následující dva adresáře mimo kořenový adresář dokumentů a změnit názvy složek: %s
  6. Vytvořte (pokud ještě neexistují) a zpřístupněte adresáře pro zápis: %s
  7. Zapněte cookies a JavaScript ve vašem prohlížeči.

Lokální instalace

Pokud provozujete lokální prostředí pro vývoj nebo testování, ujistěte se, že splňujete předchozí požadavky. Jakmile je to hotové, zkopírujte obsah adresáře htdocs (z distribučního souboru XOOPS) do kořenové cesty dokumentů vašeho webového prostředí. Jakmile jsou soubory zkopírovány, můžete spustit instalaci zadáním adresy "http://vasestranka.cz" do prohlížeče. Tím se spustí proces instalace.

Instalace na hostované platformě

Pokud provozujete prostředí na hostované platformě, rozbalte soubory XOOPS lokálně nebo na serveru, pokud máte přístup přes shell (např. SSH). Jakmile je to hotové, ujistěte se, že přesunete nebo zkopírujete všechny soubory XOOPS z adresáře htdocs do kořenového webového adresáře (váš poskytovatel obvykle poskytne tuto cestu s pokyny). Jakmile jsou soubory zkopírovány, můžete spustit instalaci zadáním adresy "http://vasestranka.cz" do prohlížeče. Tím se spustí proces instalace.

Pokračování v instalaci

Postupujte podle krok za krokem uvedených pokynů v instalátoru.
